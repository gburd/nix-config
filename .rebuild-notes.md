# Rebuild Notes - Rust Configuration Fix

## Changes Made

Fixed package conflicts in both pinentry and Rust configurations.

### Fixed Issues

1. **Pinentry conflict** - Multiple pinentry packages providing `/bin/pinentry`
2. **Rust conflict** - Both `rustup` and standalone `cargo` providing `/bin/cargo`

### Solutions

1. **Pinentry**: Use absolute paths in wrapper, only install wrapper + gcr library
2. **Rust**: Use rustup exclusively, remove standalone cargo/rustc packages

## After Rebuilding

Once you run `home-manager switch --flake .#gburd@floki`, you'll need to initialize rustup:

```bash
# Set stable as default toolchain (this installs rust and makes cargo/rustc available)
rustup default stable

# Add useful components
rustup component add clippy rustfmt rust-src llvm-tools-preview

# Verify everything works
rustc --version
cargo --version
cargo-llvm-cov --version
```
