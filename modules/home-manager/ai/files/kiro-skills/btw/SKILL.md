---
name: btw
description: Quick aside or observation without derailing the current task. Use when the user says "btw" or wants to note something without changing context.
---

The user has a quick aside. Handle it without losing the thread of whatever you were doing.

Rules:
- Answer the aside briefly and directly
- Do NOT start a new task or change what you were working on
- If it requires action, note it as a TODO but keep working on the current task
- If it's informational, acknowledge and continue
